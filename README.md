# EBFI - Eryk's Brainfuck Interpreter

## Compiling
```bash
$ cmake .
$ make
```

## Usage
```bash
$ ./ebfi code_file
```
For example
```bash
$ ./ebfi examples/hello_world.bf
```

## Examples
- hello_world.bf - https://pl.wikipedia.org/wiki/Brainfuck
- 99_bottles.bf - http://www.99-bottles-of-beer.net/language-brainfuck-101.html
